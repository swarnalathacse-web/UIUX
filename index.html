UI/UX Reference

Design tools that move fast

A quick guide to the most-used tools in modern UI/UX workflows — from wireframing to handoff.

Popular Tools
🎨
Figma
Design & Prototyping

Browser-based, real-time collaborative design tool. Industry standard for UI design, components, and dev handoff.

Collaboration Prototyping Free tier
🖊️
Adobe XD
Design & Prototype

Adobe's vector-based UX tool for designing and prototyping apps and websites with voice prototyping support.

Adobe Suite Voice UI
📐
Sketch
UI Design

Mac-native design tool known for its symbol system and plugin ecosystem. Popular in product and app design teams.

Mac only Plugins Symbols
🗂️
Notion / FigJam
Ideation & Whiteboard

FigJam for collaborative whiteboarding and sticky notes; Notion for organizing research, user journeys, and documentation.

Whiteboard Research Docs
🔬
Maze / Useberry
User Testing

Run unmoderated usability tests on prototypes. Gather click maps, heatmaps, and task completion rates from real users.

Usability Heatmaps Analytics
🚀
Zeplin / Storybook
Dev Handoff

Bridge design and development. Zeplin exports specs, assets, and styles; Storybook documents live UI components.

Handoff Dev-ready Components
💡 Pro tip: Start with wireframes in FigJam → refine in Figma → test with Maze → hand off via Zeplin. That's a full UX cycle.
